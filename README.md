# Travel Recommendation System

A **simple travel destination recommender** built with Python and Flask.

---

## Features
- Enter your preferences (destination, state, type, etc.)
- Get **personalized recommendations** using machine learning
- Predicts **popularity score** of your selected place
- Clean, responsive web interface

---

## How to Run

```bash
# 1. Clone the project
git clone https://github.com/ParthSavla2345/Intelligent-Travel-Recommendation-System.git
cd travel-recommender

# 2. Install dependencies
pip install flask pandas scikit-learn numpy

# 3. Run the app
python app.py
